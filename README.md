# Opencv
檔案編譯方法
<pre>
編譯 $g++ 檔案名稱 -o 執行檔名稱 `pkg-config --cflags --libs opencv`
執行 $sudo ./執行檔名稱
</pre>

<b>image_to_gray</b>
<br>圖片轉灰階</br>

<b>opencv_webcam</b>
<br>打開視訊鏡頭</br>

<b>color_track</b>
<br>顏色追蹤</br>

<b>image_depth</b>
<br>顏色深度</br>

<b>image_sharpen</b>
<br>圖片銳化</br>

<b>image_add</b>
<br>圖片相加</br>
